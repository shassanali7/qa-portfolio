# Critical Bugs

This folder contains high-severity bugs identified during testing.

## Definition
Critical bugs are issues that:
- Break core functionality
- Cause data loss
- Block user actions
- Affect multiple users

## Bug List
Each file contains:
- Steps to reproduce
- Expected result
- Actual result
- Severity
- Environment
