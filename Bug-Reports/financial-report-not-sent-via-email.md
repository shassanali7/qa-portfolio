# Bug: Financial Report Not Sent via Email

## Module
Financial Reports

## Platforms
- Android
- iOS

## Description
The application fails to send generated financial reports via email. This prevents users from accessing reports externally.

## Steps to Reproduce
1. Navigate to Financial Reports  
2. Generate a report  
3. Select the option to send via email  
4. Confirm action  

## Expected Result
- Financial report should be sent to the user's email  

## Actual Result
- Report is not sent  
- No confirmation or failure message is shown  

## Severity
High

## Priority
High

## Impact
- Core feature failure  
- Users cannot share or store reports externally  
- Affects usability and reliability  

## Notes
Check email service integration or backend API failure
