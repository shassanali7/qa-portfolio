# Bug: Continuous Loader Appears While Scrolling Leads Page

## Module
Leads

## Platform
- Mobile App

## Description
The Leads page continuously triggers a loader when scrolling, affecting usability and performance.

## Steps to Reproduce
1. Navigate to Leads page  
2. Scroll through the list  

## Expected Result
- Smooth scrolling experience  
- Loader should appear only when fetching new data  

## Actual Result
- Loader appears repeatedly during scrolling  

## Severity
High

## Priority
High

## Impact
- Poor user experience  
- Performance degradation  
- Interrupts browsing  

## Notes
Possible issue with pagination or infinite scroll logic
