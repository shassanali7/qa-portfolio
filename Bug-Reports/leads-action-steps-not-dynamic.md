# Bug: Same Action Steps Displayed for All Service Types in Leads

## Module
Vendor Dashboard → Leads

## Platform
- Mobile App

## Description
Leads display identical action steps across all service types instead of showing service-specific steps.

## Steps to Reproduce
1. Navigate to Leads section  
2. Open leads with different service types  
3. Review Action Steps  

## Expected Result
- Each service type should display unique and relevant action steps  

## Actual Result
- All service types show identical action steps  

## Severity
Medium

## Priority
Medium

## Impact
- Reduces usefulness of leads  
- Confuses vendors  
- Incorrect workflow guidance  

## Notes
Service-based logic or mapping not implemented correctly
