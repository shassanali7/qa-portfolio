# Bug: Unable to Switch Between Groups

## Module
Dashboard / Group Management

## Platform
Mobile App

## Description
Users are unable to switch between different groups from the group switch feature.

## Steps to Reproduce
1. Navigate to Dashboard  
2. Open Group Switch option  
3. Select another group  

## Expected Result
- User should switch to selected group  
- Data should update accordingly  

## Actual Result
- Group does not switch  
- UI remains on current group  

## Severity
High

## Priority
High

## Impact
- Multi-group functionality broken  
- Users cannot manage multiple groups  

## Notes
Possible issue with state management or navigation handling
