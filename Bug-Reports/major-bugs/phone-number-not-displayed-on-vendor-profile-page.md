## Bug: Phone Number Not Displayed on Vendor Profile Page

## Module
Vendor Profile

## Platform
- Mobile App

## Description
The phone number field is not visible on the Vendor Profile page. However, it is available and editable in the Edit Profile screen.

## Steps to Reproduce
1. Login as Vendor
2. Fill profile form and update details
3. Navigate to Profile Page
4. Observe displayed information
5. Tap on Edit Profile

## Expected Result
- Phone number should be visible on the Profile page

## Actual Result
- Phone number is missing on Profile page
- Field is visible only in Edit Profile

## Severity
Major

## Priority
High

## Impact
- Incomplete user profile display
- Affects communication visibility

## Notes
UI may not be binding phone number field from backend response
