## Bug: Large Image Upload Fails for Projects

# Module
Project Management

# Platform
Web

# Description
Large image files fail to upload while smaller ones succeed.

# Steps to Reproduce
1. Open any project
2. Upload large image file

# Expected Result
Images within allowed size should upload

# Actual Result
Upload fails or does not complete

# Severity
Critical

# Priority
High

# Impact
Blocks project setup
Affects core functionality
