## Bug: Incorrect Username Displayed in Sidebar

# Module
Admin Panel / Sidebar

# Platform
Web

# Description
Sidebar intermittently displays an incorrect or random username instead of the logged-in admin’s name.

# Steps to Reproduce
1. Open the application
2. Login with valid credentials
3. Observe sidebar username
4. Repeat login/logout multiple times

# Expected Result
Sidebar should display correct logged-in username consistently

# Actual Result
- Incorrect or random username is displayed
- Sometimes shows previous session data

# Severity
Low

# Priority
Medium

# Impact
- Confusing identity display
- Reduces trust in system

# Notes
Possible session or caching issue
