# ECO UNITY - QA Bug Reports

This folder contains categorized bug reports identified during manual QA testing of the ECO UNITY application.

## Bug Categories

- Critical Bugs → System-breaking issues (e.g., mission not loading, app crash)
- Functional Bugs → Features not working as expected (e.g., mission completion not updating points)
- Major Bugs → Important issues affecting user experience (e.g., leaderboard not updating correctly)
- Minor Bugs → UI/UX or low-impact issues (e.g., alignment, icon issues, text inconsistencies)

## Platforms Tested
- Android
- iOS
- Web (if admin panel tested)

## Modules Covered
- Authentication (Login/Signup)
- Missions (Daily Tasks, Eco Activities)
- Leaderboard
- Profile Management
- Dashboard
- Map / Location Features

## Tools Used
- Manual Testing
- MS TEAM, JIRA or ClickUp
- Postman (if API tested)
