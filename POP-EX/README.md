# PopEx - QA Bug Reports

This folder contains categorized bug reports identified during manual QA testing of the PopEx application.

## Bug Categories

- Critical Bugs → System-breaking issues
- Functional Bugs → Features not working as expected
- Major Bugs → Important issues affecting user experience
- Minor Bugs → UI/UX or low-impact issues

## Platforms Tested
- Web
- Mobile (if applicable)

## Tools Used
- Manual Testing
- MS TEAM, JIRA or ClickUp
- Postman
