# Bug: Update Button Not Functional in Vendor Signup

## Module
Vendor Signup

## Platform
- iOS

## Description
The "Update" button in the Vendor Signup flow is not responsive and does not perform any action when tapped, blocking users from proceeding.

## Steps to Reproduce
1. Open the mobile application  
2. Sign up as a Vendor user  
3. Fill in all required fields  
4. Tap on the "Update" button  

## Expected Result
- Form should be submitted successfully  
- User should proceed to the next step or data should be saved  

## Actual Result
- Button is not responsive  
- No action is triggered  

## Severity
High

## Priority
High

## Impact
- Blocks vendor onboarding completely  
- Prevents form submission  

## Notes
Possible issue with button action binding or API trigger
