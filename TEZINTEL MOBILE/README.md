# Tezintel Mobile - QA Bug Reports

This folder contains categorized bug reports identified during manual QA testing of the Tezintel Mobile application.

## Bug Categories

- Critical Bugs → System-breaking issues
- Functional Bugs → Features not working as expected
- Major Bugs → Important issues affecting user experience
- Minor Bugs → UI/UX or low-impact issues

## Platforms Tested
- Android
- iOS

## Tools Used
- Manual Testing
- MS TEAM, JIRA or ClickUp
- Postman
