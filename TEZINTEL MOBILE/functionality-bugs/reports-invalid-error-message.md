# Bug: Invalid Error Message Displayed on Reports Page After Signup

## Module
Reports

## Platform
- Android

## Description
An incorrect or irrelevant error message is displayed on the Reports page after a new user signs up, even when the page loads successfully.

## Steps to Reproduce
1. Sign up as a new user  
2. Navigate to the Reports page  

## Expected Result
- No error message should appear if the page loads correctly  

## Actual Result
- An irrelevant or incorrect error message is displayed  

## Severity
Medium

## Priority
Medium

## Repro Rate
Needs Confirmation (Always / Intermittent)

## Impact
- Confuses users  
- Reduces trust in system feedback  

## Notes
Possible improper error handling or leftover API response message
