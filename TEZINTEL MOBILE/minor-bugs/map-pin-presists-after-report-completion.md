## Bug: Map Pin Persists After Report Completion

# Module
Report Form

# Platforms
iOS

# Description

# Steps to reproduce
1. After report generation or canceling the report 
2. Reopen the Report form 2nd page 
3. Notice the pin point in the map

# Impact
