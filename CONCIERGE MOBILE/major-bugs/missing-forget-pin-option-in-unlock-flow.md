## Bug: Missing “Forget PIN?” Option in Unlock Flow

# Module
Documents → Unlock

# Platform
iOS

# Description
The “Forget PIN?” option is missing from the document unlock flow, leaving users without a recovery option if they forget their PIN.

# Steps to Reproduce
1. Login to the app
2. Navigate to Documents
3. Attempt to unlock a document

# Expected Result
* “Forget PIN?” option should be available

# Actual Result
* No recovery option shown

# Severity
Major

# Priority
Medium

# Impact
* Users may get permanently locked out of documents
* Poor user experience and support dependency

# Notes
Recovery mechanism missing in UI
