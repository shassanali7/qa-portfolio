## Bug: Missing Distribution Graph in Project Summary

# Module
View Project → Summary

# Platform
iOS

# Description
The Distribution Graph is missing from the Project Summary page in the iOS application. This graph is available and functioning correctly in the web version but is not displayed on mobile.

# Steps to Reproduce
1. Login to the iOS app
2. Navigate to View Project
3. Open the Summary section

# Expected Result
* Distribution Graph should be visible and display relevant data

# Actual Result
* Graph is not displayed on iOS
* Section appears incomplete

# Severity
Major

# Priority
Medium

# Impact
* Users cannot visualize distribution data
* Reduced feature parity between web and mobile

# Notes
Possible missing API integration or UI rendering issue
