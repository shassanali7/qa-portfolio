## Bug: Distribution Graph Missing Year Filter

# Module
View Project → Summary (Graph Section)

# Platform
iOS

# Description
The Distribution Graph does not provide a year-based filtering option, limiting the user’s ability to analyze data over time.

# Steps to Reproduce
1. Login to the iOS app
2. Navigate to View Project → Summary
3. Locate Distribution Graph

# Expected Result
* User should be able to filter graph data by year

# Actual Result
* No filter option available

# Severity
Major

# Priority
Medium

# Impact
* Limited data analysis capability
* Reduced usability for financial insights

# Notes
Feature exists on web → likely missing implementation on iOS
